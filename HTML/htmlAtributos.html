Atributo Href

O link do atributo href (abreviação de “Hypertext REFerence”) indica a relação entre diferentes páginas para os mecanismos de pesquisa. O href é um atributo da tag âncora e contém dois componentes: A URL (o link real); O texto ou objeto clicável que os usuários verão na página (conhecido como “texto âncora”)

O atributo Alternativo

O atributo alt deve refletir o conteúdo da imagem, para que os usuários que não conseguem ver a imagem entendam o que ela contém:

O texto alternativo, também conhecido como alt text, alt tags ou descrições alt, é utilizado para descrever o conteúdo de uma imagem. Ele aparece no lugar da imagem quando ela não consegue ser carregada. Ainda, ele permite que os mecanismos de pesquisa rastreiem e classifiquem seu site de uma melhor forma.
