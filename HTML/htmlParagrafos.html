A tag "p" permite a criação de paragrafos

Este parágrafo contém muitas linhas no código-fonte, mas o navegador ignora isso.

This paragraph contains a lot of spaces in the source code, but the browser ignores it. Esse paragrafo contém muitos espaços no código-fonte, mas o navegador ignora isso.

O número de linhas em um parágrafo depende do tamanho da janela do navegador. Se você redimensionar a janela do navegador, o número de linhas neste parágrafo mudará.

No HTML espaços entre as linhas são ignorados:

Minha Bonnie está sobre o oceano. Minha Bonnie está sobre o mar. Minha Bonnie está sobre o oceano. Oh, traga minha Bonnie de volta para mim.

A teg que preserva espaços e quebra de linhas: "pre"

        Minha Bonnie está sobre o oceano.

        Minha Bonnie está sobre o mar.

        Minha Bonnie está sobre o oceano.
